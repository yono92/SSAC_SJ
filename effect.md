### CSS Position

https://www.w3schools.com/css/css_positioning.asp

> position 속성값
>
> - absolute : 문서에 포함되기 때문에 스크롤시 같이 움직임
> - fixed : 문서에 포함되지 않기 때문에 스크롤시 고정됨
>
> - 좌표로 위치를 설정 : top, right, bottom, left 키워드
> - 박스와 박스를 포갤 수 있음.
> - 포개진 박스의 순서를 설정할 수 있음
