### HTML Element
https://www.w3schools.com/html/html_elements.asp

> 포함관계
> -기준에 따라서 조상, 조상요소(Acestor), 부모요소(Parent), 자식요소(Child), 자손요소(Descendant)

```
<html>
  <body>
    <h1>Web Page</h1>
  </body>
</html>

* html : body의 부모요소, h1의 조상요소 

* body : html의 자식요소, h1의 조상요소

* h1 : html의 자손요소, body의 자식요소 

```
